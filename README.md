[cite_start]MIST is a next-generation social media platform designed to solve authenticity and trust issues in the current market[cite: 1, 4]. 

### Key Features:
* [cite_start]**Authenticity Flow**: Two-step account creation with AI-monitored selfie verification[cite: 15, 17].
* [cite_start]**Karma Point System**: A gamified engine where users earn reach and monetization based on positive contributions and reporting fake content[cite: 55, 58, 62].
* [cite_start]**Local Bubbles**: Radius-based content and anonymous voice chat rooms for localized discussions[cite: 47, 65, 66].
* [cite_start]**Data Sovereignty**: All data is stored and regulated within India[cite: 51, 53].

### Tech Stack (MVP):
* **Frontend**: Flutter
* **Backend**: Supabase (Mumbai Region)
* **AI**: Google ML Kit (On-device)
